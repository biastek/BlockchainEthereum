# etherRopsten-app

## Features:
- [x] Login/Register.
- [x] Manage Accounts.
- [x] getBalance/transferFrom/createAccount/Token.
- [x] transferEther/Ether.
- [x] faucet eth/token.


## Installation

1. Clone the Repo.
2. Config Rest-API.
2. Config Database.
3. Go into the folder and hit the npm install command
4. Run the command: npm start
